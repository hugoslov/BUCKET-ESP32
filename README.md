# BUCKET / BALDE-ESP32

Sistema eletrônico da composteira inteligente BUCKET, desenvolvido com ESP32 e sensores para monitoramento da compostagem.

## Componentes utilizados

- ESP32
- Sensor DS18B20 à prova d’água
- Sensor capacitivo de umidade V2
- Sensor de pH RS485

## Funcionamento

Os sensores coletam dados da composteira, como temperatura, umidade e pH.  
Essas informações são processadas pelo ESP32 e enviadas para o aplicativo BUCKET, permitindo o acompanhamento da compostagem em tempo real.

## Objetivo

Facilitar o processo de compostagem, ajudar no reaproveitamento de resíduos orgânicos e incentivar práticas sustentáveis.
